# DWT module specifications

Available wavelets
------------------

+ Haar
+ Daubechies
  - db1, db2, db3, ..., db10
+ Symlets
  - 2, 3, ..., 8
+ Coiflets
  - 1, 2, ..., 5
+ Biorthogonal
  - 1.1, 1.3, 1.5, 2.2, 2.4, 2.6, 2.8, 3.1, 3.3, 3.5, 3.7, 3.9, 4.4, 5.5, 6.8
+ R_Biorthigonal

They must be calculated from 1 to 10 levels.
